# Linux Basic Commands Assignment

## Overview
This repository contains my Linux basic commands assignment demonstrating:

- File and directory management
- Viewing file contents
- Searching using grep
- Zipping and unzipping files
- Downloading files using wget
- File permission management
- Environment variable usage

## Commands Covered

### 1. File & Directory Operations
- mkdir
- touch
- mv

### 2. File Viewing
- cat
- head
- tail

### 3. Pattern Searching
- grep

### 4. Compression
- zip
- unzip

### 5. Downloading Files
- wget

### 6. Permissions
- chmod

### 7. Environment Variables
- export
